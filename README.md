# Learn to Code with Rust

Rust is a general purpose, **statically typed** programming language focused on **performance**, **safety** and **concurrency**. Created by **Graydon Hoare** in **2006** while at **Mozilla**, it was officially sponsored by Mozilla in 2009 and reached its first stable release, **1.0** in **May 2015**. Since February 2021, the project has been supported by the **Rust Foundation**, a nonprofit backed by companies including **Google**, **Microsoft**, **AWS**, **Huawei** and **Mozilla**.

The Rust power that makes it different from other languages is that you will get **memory safety** without having a **Garbage collector**
